Esta aplicação foi desenvolvida utilizando o Framework Django da linguagem de Programação Python.

Para inciar o projeto, utilize os scripts abaixo:
* Mais detalhes sobre estes scripts podem ser encontrados no documento principal do Auto Instrucional.
1 - build_app_ai.sh : Irá instanciar o ambiente e configurar a aplicação (Obrigatório)
2 - create_initial_data.sh : Adiciona dados fictícios na aplicação para visualizar seu funcionamento. (Opcional)
3 - clean_database.sh : Remove todos os dados do banco de dados para "reiniciar" a aplicação. (Opcional)
4 - start_app.sh : Inicia a aplicação na endereço localhost:8000. (Obrigatório)
